// Features

Everything you need to chat

No sign-up friction. Jump in with a room code or create a permanent account.

🔐
PERMANENT LOGIN
Email-based auth via Firebase. Same email = same room, always. Stay logged in across sessions.
🎭
ANONYMOUS MODE
Pick a username and room code — no account needed. Optional room passwords for privacy.
⚡
REAL-TIME SYNC
Messages sync instantly across all connected users via Firebase Realtime Database.
🖼️
IMAGE SHARING
Send images up to 5MB directly in chat. Full-screen viewer with zoom and pan.
😊
EMOJI & GIF
Built-in emoji picker, GIF search, message reactions, and reply/quote support.
👑
ADMIN CONTROLS
Room creator gets admin badge and can delete any message to moderate the room.
