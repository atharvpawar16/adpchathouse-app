# Contributing to ADP Chathouse

Thank you for your interest in ADP Chathouse. While this is primarily a personal project, contributions in the form of bug reports, suggestions, and forks are welcome.

---

## Reporting Bugs

If you find a bug, please open an issue and include:

- A clear and descriptive title
- Steps to reproduce the issue
- Expected behavior vs. actual behavior
- Your Android version and device model
- Screenshots or screen recordings if applicable

---

## Suggesting Features

Have an idea? Open an issue with the label `enhancement` and describe:

- What the feature does
- Why it would be useful
- Any implementation ideas you have in mind

---

## Forking & Building Your Own Version

You are welcome to fork this repository and build your own version. To get started:

1. Fork the repo on GitHub
2. Clone your fork locally
   ```bash
   git clone https://github.com/your-username/adpchathouse-app.git
   ```
3. Set up your own Firebase project and add `google-services.json` to `app/`
4. Update the Firebase config in `app/src/main/assets/index.html`
5. Build and run in Android Studio

---

## Pull Requests

Pull requests are not actively reviewed or merged at this time, as this is a personal project. If you have a meaningful fix or improvement, feel free to open a PR anyway — it may be considered in the future.

---

## Code Style

If you do submit a PR, please follow these guidelines:

- Use consistent indentation (4 spaces for Java, 2 spaces for HTML/CSS/JS)
- Write clear, descriptive commit messages
- Keep changes focused — one feature or fix per PR
- Comment non-obvious logic
